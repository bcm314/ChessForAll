ChessForAll - это шахматная программа с мощным шахматным движком Stockfish и многими полезными функциями.

Функции
  • Игра человека с компьютером
  • Игра двух человек
  • Игра компьютера с компьютером
  • Анализ позиции
  • Редактор позиции
  • Собственная база партий (.pgn-db)
  • Сохранение, вставка, изменение и удаление партии
  • Установка позиций
  • Копирование в буфер обмена (партии или позиции)
  • Книга дебютов
  • Выбор силы игры
  • Настройка часов
  • Портрет / альбомный режим

Инструкция по эксплуатации

Вызов основного меню: Проведите пальцем с левого края доски направо

Вызов игрового меню: Проведите пальцем с правого края экрана налево

Вызов меню настройки доски: Долгое нажатие пальцем на доску и отпустите

Управление доской: движение пальца по доске
  • Следующий ход: направо
  • Предыдущий ход: налево
  • Начальная позиция: в левый угол доски
  • Конечная позиция: в правый угол доски

Кнопки быстрого старта (в нижнем правом углу слева направо):

  • Настройка режимов игры
  • Старт/Стоп анализа
  • Выполнить ход компьютером
  • Удалить ход
  • Повернуть доску
  • Предыдущий ход
  • Следующий ход

Контекстное меню при долгом нажатии и отпускании

  • Настройки
  • -
  • Автоматическая переигровка партиии
  • -
  • -
  • Первый ход
  • Последний ход

Управление доской: движение пальца по доске (при загрузке партии из текущего файла PGN)

  • Следующая партия : направо
  • Предыдущая партия:  налево
  • Первая партия: в левый угол
  • Последняя партия: в правый угол
  • Случайная партия: от последней кнопки к первой кнопке
  • Поиск партии: от первой кнопки к последней кнопке

Нажатие кнопки Имя или Эло: Редактирование данных о партии

Нажатие кнопки Компьютерный варианты Настройки компьютера-движка

Нажатие поля Время: Изменение текущего времени; долгое нажатие: Установка настроек часов

PGN файлы

Доступны следующие функции:

  • Загрузка партии
  • Сохранение, вставка, замена и удаление партии
  • Загрузка файла PGN
  • Копирование партии в буфер обмена
  • Копирование позиции в буфер обмена
  • Вставка партии из буфера обмена

Долгое нажатие на файле PGN для его удаления (.pgn) и для удаленияфайла базы данных (.pgn-db).

Файл PGN может быть также загружениз менеджера файлов.

База данных партий (.pgn-db)

Для игры программа использует свою собственную шахматную базу данных (файл: .pgn-db). Если файл базы данныхпартий отсутствует при загрузке / сохранении, он будет создан. В строке состояния отображается текущий статус создания базы данных. Во время этого процесса файл PGN заблокирован. Другие действия могут быть предприняты без каких-либо проблем.

Использование базы данных PGN можно отключить в настройках. В этом режиме игра всегда добавляется в конец файла.

Книга дебютов

Небольшая книга дебютов уже интегрирована в ChessForAll. Однако, внешнюю более обширную книгу дебютов (.bin) также можно использовать. Управление книгами дебютов доступно в режиме настройки игры компьютера, вызываемого из: Основное меню/Компьютер или при долгом нажатии пальцем на поле Компьютерные варианты.

Установка пользовательских настроек

  • Имя игрока
  • Показывать полосу статуса
  • Менять сторону после хода
  • Показывать последнюю позицию (загружать)
  • Убрать экран окончания времени
  • Навигация попозиции (доске)
  • Использовать PGN базу
  • Звуковые эффекты
  • Координаты (1-h, 1-8)

Настройки  движка Stockfish

  • Автоматически стартовать при входе в программу
  • Показывать варианты
  • 1й ход (белые): Случайный ход
  • Активация/деактивация расширенной книги дебютов
  • Расширеннная книга дебютов: Загрузить внешний .bin файл
  • Копьютерные варианты: Количество вариантоа и ходов
  • Сила игры (0...100)
  • Уровень атаки (0...100), защиты (0...-100)
  • Запись логов (сообщений движка)
